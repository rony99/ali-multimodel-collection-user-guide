## FAQ

**Q：可以用自己的模型 / 自己的 API 吗？**  
A：不可以。**我们会提供 API**（及连接配置），须用平台指定的三套模型采集，不能换成自备 API 或本地模型。

**Q：怎么才算「通过」？**  
A：单次看 Docker 里跑 `tests/test.sh` 是否退出码 `0`。题目是否最终合格，还要看操作步骤里的汇总门槛，并由后续评定。

**Q：「一条数据」是某一个模型跑一次吗？**  
A：不是。一条 = **一道完整题**（甲方数据包 + 三模型多次 session），不是「某一个模型跑一次」。

**Q：预检通过了就算过关吗？**  
A：不算。预检只做简单检测；最终通过与否需要后续评定。
