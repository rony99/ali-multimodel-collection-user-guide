# 参与方式

欢迎具备相关背景的同学参与本项目的 Harbor 三模型比对数据采集。请先阅读 [用户操作步骤.md](./用户操作步骤.md)，有疑问用 [qa_skill/](./qa_skill/) 答疑；确认仍有兴趣后再按下方流程报名。

## 基本要求

1. **相关从业人员**  
   面向甲方约定的理工类程序员业务方向，例如：软件工程（前端 / 后端）、数据分析、测试、运维（DevOps）等。须为相关岗位或同等从业背景。

2. **Claude Code 实战经验**  
   有 Claude Code 的实际使用经验，并曾用 Claude Code **完成过至少两个项目**（完整做过，而非仅试用过）。

3. **Docker 与单元测试**  
   有实际使用 Docker 的经验，以及编写 / 运行单元测试的经验（本采集任务依赖 Docker 跑测与 `test.sh` 判分）。

4. **已读文档并仍有兴趣**  
   已阅读 [用户操作步骤.md](./用户操作步骤.md)；部分疑问已通过 [qa_skill/](./qa_skill/) 解答，**仍然有兴趣参与**。

## 报名材料（微信发给「栗子」）

请编辑以下内容，微信发送给 **栗子**，审核通过后进入**项目参与群**：

1. 自己的**岗位 / 业务类型**（对照上方理工类程序员方向简要说明）  
2. **Claude Code 使用经验**（是否完成过 ≥2 个项目，可简述项目类型）  
3. **是否用过 Docker**、**是否写过单元测试**  
4. **每天可以参与的时间**（大致可用时段或小时数）  
5. **对采集任务还有什么问题**（没有可写「暂无」）

### 发送模板（可直接复制填写）

```text
【Harbor 采集报名】
1. 岗位/业务类型：
2. Claude Code 经验（含完成过的项目数/简述）：
3. Docker：用过 / 没用过；单元测试：写过 / 没写过：
4. 每天可参与时间：
5. 对采集任务的问题：
```
